Overview
What
Generic-Text-Adventure seeks to provide a simple way to create a text adventure game given a series of text documents and pointers within them (or something like that). This is inspired by Twine 2.
Why
It is easy to create a single text adventure, where the branching paths and decisions are hard-coded into the file. However, creating a system that is able to generate any text adventure with user input is a more difficult task.
With What
This will only be made with Javascript, HTML, CSS, etc.
